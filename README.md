# CUBIX
Краткое резюме:
Суть: Сайт автоматически подстраивается под размер экрана 
(компьютер, планшет, смартфон), сохраняя функциональность и внешний вид.
Цель: Обеспечение оптимального пользовательского опыта (UX/UI) на всех устройствах.
Инструкции для запуска:
1. npm i (установка зависимостей)
2.npm run start (запуск проекта)

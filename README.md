# CUBIX
Полностью адаптивеня верска под любые устройстваю 
Сайт автоматически подстраивается под нужный размер экрана, 
обеспечивая оптимальный пользовательский ul на всех устройствах.
Сайт сохраняет свою функциональность, читаемость и внешний вид независимо от устройства, 
будь то настольный компбютер, ноутбук, планшет или смартфон
Для локалного запуска
1. npm i
2. npm run start
